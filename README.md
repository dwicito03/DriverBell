# DriverBell v1.7.2

DriverBell adalah software pemutaran bell sekolah otomatis untuk Windows.
Aplikasi ini berjalan offline, ringan, dan dirancang agar mudah dioperasikan termasuk dengan screen reader.

## Fitur Utama
- Jadwal bell otomatis berdasarkan jam dan hari.
- Status bell aktif/nonaktif per entry.
- Kolom `Last Played` (terakhir diputar) dengan format waktu sesuai bahasa aplikasi.
- Pratinjau audio + tombol hentikan pratinjau.
- Dukungan audio multi-format (contoh: `wav`, `mp3`, `aac`, `ogg`, `flac`, `m4a`).
- Pemilihan perangkat keluaran audio (`Output Device`) dari menu Pengaturan.
- Welcome dialog yang bisa diaktifkan/nonaktifkan saat startup.
- Menu About + dialog Donasi (Bank Jago, GoPay, PayPal).

## Instalasi
1. Gunakan Windows 64-bit.
2. Jalankan `DriverBell.exe` atau installer sesuai paket rilis.
3. Jika muncul peringatan SmartScreen, pilih `More info` lalu `Run anyway`.

## Cara Pakai Singkat
1. Buka DriverBell.
2. Klik `Tambahkan Bell`.
3. Isi nama, jam, hari, dan pilih file audio.
4. (Opsional) Coba `Pratinjau Audio`.
5. Simpan lalu `Aktifkan` entry.

## Catatan
- Konfigurasi dan log disimpan di `%APPDATA%\DriverBell`.
- Untuk rilis yang memakai backend BASS, pastikan `bass.dll` tersedia dalam paket aplikasi.
