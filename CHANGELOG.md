# Changelog

## DriverBell v1.7.2 (2026-02-23)

### Perubahan Besar
- Penyegaran tampilan aplikasi (UI) dengan nuansa sekolah, ikon aksi yang lebih jelas, serta ikon brand baru untuk aplikasi dan tray.
- Penambahan kolom dan data `Last Played` pada jadwal bell, termasuk format tanggal/waktu yang menyesuaikan bahasa pengguna.
- Penambahan menu `About` dengan informasi produk versi 1.7.2 dan integrasi tombol `Donasi Sekarang`.
- Perombakan dialog donasi menjadi multi-metode (Bank Jago, GoPay, PayPal) dengan tombol salin per akun.
- Penambahan pengaturan untuk menampilkan/menyembunyikan welcome dialog saat startup.
- Penambahan fitur `Pratinjau Audio` dan `Hentikan Pratinjau` pada form tambah/edit bell.
- Migrasi backend audio ke `BASS` untuk playback yang lebih stabil pada desktop Windows.
- Dukungan audio multi-format untuk bell dan pratinjau (misalnya `wav`, `mp3`, `aac`, `ogg`, `flac`, `m4a`).
- Penambahan pengaturan `Output Device / Perangkat Keluaran` agar pengguna dapat memilih perangkat audio tujuan.
- Penyempurnaan aksesibilitas dan navigasi keyboard (termasuk alur fokus yang lebih stabil untuk pengguna screen reader/NVDA).
